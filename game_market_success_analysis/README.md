# Оптимизация маркетинговых затрат в сервисе по продаже билетов
### Описание:
Нас пригласили на стажировку в отдел аналитики сервиса оп продаже билетов. Задача: помочь маркетологам оптимизировать маркетинговые затраты. В распоряжении есть данные от сервиса с июня 2017 по конец мая 2018 года: лог сервера с данными о посещениях сайта Яндекс.Афиши, выгрузка всех заказов за этот период, статистика рекламных расходов.
### Используемые инструменты
* Pandas - библиотека для обработки и анализа данных
* NumPy - модуль, который предоставляет общие математические и числовые операции в виде пре-скомпилированных, быстрых функций
* SciPy - библиотека д предназначенная для выполнения научных и инженерных расчётов
* Matplotlib - библиотека для визуализации данных
* Seaborn - бибилиотека для визуализации данных
