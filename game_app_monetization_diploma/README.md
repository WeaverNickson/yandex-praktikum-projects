# Формирование модели монетизации игрового приложения
### Описание:
Основной датасет содержит данные о событиях, совершенных в мобильной игре ~~название засекреченно~~. В ней пользователи строят свою космическую программу и пытаются преуспеть в нелёгком деле колонизации галактики. Основная монетизация игры — только планируется. Но предполагается, что в приложении будет происходить показ рекламы на экране с выбором типа объекта для постройки. 
### Используемые инструменты
* Pandas - библиотека для обработки и анализа данных
* NumPy - модуль, который предоставляет общие математические и числовые операции в виде пре-скомпилированных, быстрых функций
* SciPy - библиотека д предназначенная для выполнения научных и инженерных расчётов
* Matplotlib - библиотека для визуализации данных
* Seaborn - бибилиотека для визуализации данных
